# Mein Lebensrad – Vom Funktionieren zum Fühlen

**Tanja Sattelberg · Systemische Beraterin & Coach**

Ein interaktives Lebensrad-Tool als Begleitung zum Selbstlernkurs *„Vom Funktionieren zum Fühlen"*. Entwickelt für Frauen über 50, die den Weg vom Funktionieren zum wirklichen Fühlen gehen.

> **Hinweis:** Dieses Tool ist ein Coaching- und Selbstreflexionsangebot. Es ersetzt keine Psychotherapie und keine medizinische Behandlung. Bei psychischen Erkrankungen oder Krisen wende dich bitte an eine qualifizierte Fachperson.

---

## Was dieses Tool bietet

### Die 5 Tabs

| Tab | Inhalt |
|---|---|
| **Mein Start** | Eingangs-Einschätzung aller 8 Lebensbereiche mit Schiebereglern und interaktivem Rad |
| **Kursbegleitung** | Bereichsbezogene Übungen, Reflexionsfragen und Affirmationen aus den 8 Modulen – mit Vertiefungshinweisen |
| **Nach dem Kurs** | Abschluss-Einschätzung, Abschluss-Reflexion und Abschluss-Moment |
| **Mein Wachstum** | Vorher-Nachher-Vergleich mit automatisch übernommenen Kurs-Einschätzungen |
| **Nächster Schritt** | Vertiefungskurse der Reihe „Mein Veränderungsweg" und 1:1 Coaching-Angebote |

### Ergänzende Übungen im Start-Tab

- **Der Lebenskompass – Drei Dimensionen** (Modul 1): Jeder Lebensbereich wird in drei Dimensionen bewertet — Kraftlevel, Klarheit und Ressourcenzugang
- **Satzergänzungs-Übung** (Modul 2): Fünf Satzanfänge zur Bewusstmachung innerer Glaubenssätze
- **Die Grenz-Reflexion** (Modul 6): Fünf strukturierte Situationsblöcke zur Auseinandersetzung mit inneren Ja/Nein-Mustern
- **Der Übergangs-Brief** (Modul 7): Schreiben aus der Perspektive des zukünftigen Ichs

### Automatische Wachstumsmessung

Nach jeder Kursübung beantworten Teilnehmerinnen 3 Einschätzungsfragen (Skala 1–5). Daraus wird automatisch ein Entwicklungs-Delta berechnet, das in den Tab *Mein Wachstum* übernommen werden kann.

### Abschluss-Moment

Am Ende des Kurses gibt es einen eigenen Abschluss-Moment mit drei Reflexionsimpulsen und einem direkten Übergang zu „Mein Wachstum".

## Vertiefungskurse: Mein Veränderungsweg

Nach Abschluss des Grundkurses können Teilnehmerinnen gezielt in einzelnen Lebensbereichen tiefer gehen:

| Kurs | Lebensbereich |
|---|---|
| Mehr bei mir | Inneres Erleben |
| Mehr ich | Beziehungen |
| Mehr Freiheit | Arbeit |
| Mehr Klarheit | Finanzen |
| Mehr Heimat | Körper |
| Rauchfrei | Körper (Vertiefung) |
| Mehr Mut | Entwicklung |
| Schritt für Schritt | Alltag |
| Mehr Leichtigkeit | Lebensfreude |

## Technisch

- Reines HTML/CSS/JavaScript — keine externen Abhängigkeiten außer Google Fonts
- Speicherung aller Werte und Texteingaben im `localStorage` des Browsers (kein Server, keine Daten verlassen das Gerät)
- Funktioniert offline und als GitHub Pages Seite
- Optimiert für iPhone/iOS (Touch-Events, Safe Area, kein Scroll-Flickern)
- Druckfunktion / PDF-Export integriert
